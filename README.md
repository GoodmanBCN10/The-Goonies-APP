# The Goonies Installer

Instalador inteligente para Nintendo Switch.
Desarrollado para la comunidad Switch ES - The Goonies OS por **GoodmanBCN**.

## Características Principales

*   **Instalador de Juegos:** Soporte para instalar de forma fácil tus juegos y aplicaciones.
*   **Instalación por MTP:** Conecta tu consola al PC mediante un cable USB para transferir e instalar contenido directamente sin sacar la microSD.
*   **Gestor de Partidas Guardadas (Saves):** Haz copias de seguridad, restaura y gestiona tus progresos.
*   **Gestor de Juegos Instalados:** Visualiza el almacenamiento que ocupa cada título y bórralos cómodamente.
*   **Gestor de Homebrew:** Controla tus aplicaciones `.nro`.
*   **Explorador de Archivos:** Navega por los archivos de tu tarjeta microSD.
